HR Dashboard (Power BI)

This project showcases an interactive HR Dashboard built using Power BI to analyze and visualize key workforce metrics. The dashboard provides a comprehensive overview of employee data, including total employees, gender distribution, salary overview, citizenship status, marital status, and employee positions.

Dynamic charts, donut visuals, and summary cards are used to present insights clearly and intuitively, enabling HR teams and management to make data-driven decisions efficiently. The dashboard improves visibility into workforce composition, supports quick analysis, and enhances overall employee management clarity.

Key Features:

Total employee count with gender breakdown

Salary overview and percentage distribution

Citizenship and marital status analysis

Employee position distribution visualization

Clean, modern, and user-friendly dashboard design

Tool Used: Power BI
